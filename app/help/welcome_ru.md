# Добро пожаловать в ProxyVault

## Короткий quick start

1. Добавьте своё подключение.
2. Выберите его в списке.
3. Нажмите `Подключить`.
4. Если нужно, нажмите `Сделать основным`.
5. Проверьте состояние в правой панели.

## Расширенная версия

ProxyVault помогает хранить подключения локально и запускать их без лишней путаницы.

Начните с одного профиля. Этого достаточно, чтобы понять весь сценарий:

- добавьте URI, WireGuard-конфиг или подписку
- выберите запись в библиотеке
- нажмите `Подключить`
- при необходимости сделайте подключение основным
- если что-то пошло не так, прочитайте короткое объяснение и только потом открывайте технический журнал

Помните о двух важных вещах:

- TCP-проверка помогает понять, отвечает ли сервер по сети, но не заменяет реальный запуск подключения.
- WireGuard живёт отдельным маршрутом и не притворяется обычным системным proxy-подключением.

Если вы переносите библиотеку на другой компьютер, сначала проверьте, что скачали архив для своей системы.
