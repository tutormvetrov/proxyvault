# Runtime License Bundle

This folder contains licensing references for third-party runtime assets that ProxyVault ships directly or bootstraps automatically.

- `SING_BOX.md` describes the bundled `sing-box` runtime.
- `WIREGUARD_WINDOWS.md` describes the official WireGuard for Windows bootstrap payload.
- `WINTUN.md` describes the Wintun component used by Windows tunnel runtimes.
- `AMNEZIAWG_WINDOWS.md` describes the bundled AmneziaWG Windows runtime files.

When an upstream project provides the canonical license text in its own repository or installer, the notice file points to that canonical source.
